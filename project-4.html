>> PROJECT_04 / MOBILE

ROAMIFY

A native Android trip planning app with destination discovery, location search, accommodation booking, booking management, and a full user account system built in Android Studio.

AndroidAndroid StudioJavaXML LayoutsMobile UI

>> PROJECT_OVERVIEW

Roamify is a native Android trip planning application built in Android Studio, designed to make travel planning effortless, visual, and enjoyable. The app combines destination discovery, accommodation search, and personal booking management into a single cohesive experience — giving travellers everything they need to plan a trip from idea to itinerary, all within a clean and intuitive mobile interface.

The Home Screen greets users with a curated feed of cool, trending, and must-visit destinations from around the world. Destinations are presented as visually rich cards featuring cover imagery, country, and a brief highlight. A More to Explore section surfaces lesser-known destinations sorted by region, theme (adventure, cultural, beach, mountain), and traveller rating — helping users discover places they might not have considered.

The Search Functionality allows users to search for any location worldwide — entering a city, country, or landmark returns relevant destination results with key travel information. The search is fast and surfacing-focused, giving users a snapshot of each location including climate, top attractions, and available accommodation options to help with initial planning decisions.

The Stay Option lets users browse accommodation options for their chosen destination — hotels, guesthouses, and rentals are listed with ratings, price ranges, and available dates. Users can select a property and proceed to a booking confirmation flow, which saves the booking to their account for future reference.

The My Bookings section provides a personal dashboard where users can view all their confirmed trip bookings — destination, dates, accommodation, and status — in a clear timeline layout. This acts as a personal travel record and upcoming trip manager. The Profile Page handles account details, preferences, and trip history, while the Authentication System supports full sign-up and login, ensuring all bookings and preferences are tied securely to the user's account.

Roamify covers the full travel planning arc in a single native app — from inspiration and destination discovery, through accommodation browsing and booking, to a personal trip management dashboard. Built entirely in Android Studio with Java and XML, it demonstrates real-world mobile UI/UX, data persistence, user authentication, and multi-screen navigation.

>> CHALLENGES

Building a smooth, visually rich scrolling experience in Android Studio — particularly the destination card feed on the home screen — required efficient RecyclerView implementation with image loading optimised to avoid jank. Designing the booking flow to be simple (minimum taps from discover to confirmed) while still capturing all necessary information required multiple UI iterations. Structuring the data layer to keep bookings, user profiles, and destination data cleanly separated and efficiently queryable was a key architectural challenge.

>> OUTCOME

Roamify delivers a genuinely useful travel companion that covers the full planning arc — from inspiration and discovery through to booking confirmation and trip management. The project demonstrates native Android development capability including UI/UX design, data persistence, user authentication, and multi-screen navigation in a real-world application context.

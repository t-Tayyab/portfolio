>> PROJECT_05 / DESKTOP

MOVIE APPLICATION

A Python desktop application built with Tkinter and a live Movie API — featuring movie browsing, real-time search, detailed film information, and an API-powered recommendations engine.

PythonTkinterMovie APIREST APIDesktop App

>> PROJECT_OVERVIEW

The Movie Application is a desktop Python program built using the Tkinter GUI library, integrated with a live Movie API to deliver a rich, interactive movie browsing and recommendation experience. The application provides a graphical interface that lets users search for films, browse popular and trending titles, explore detailed movie information, and receive personalised movie recommendations — all from a clean, native desktop window without needing a web browser.

The Browse Screen is the application's main view — on launch, it fetches and displays a grid of currently popular and trending movies from the API, each shown as a card with the movie poster, title, release year, and audience rating. The layout is scrollable and dynamically populated from live API data, meaning the content stays current every time the app is opened.

The Search Functionality allows users to type in any movie title and instantly query the API for matching results. Search results are displayed in the same card grid format with poster imagery, and selecting any result opens a detailed view for that film.

The Movie Detail View shows comprehensive information for a selected film: full synopsis, genre tags, runtime, release date, director, cast overview, and the audience rating score. This is fetched from the API in real time for accurate, up-to-date information.

The Recommendations Engine is a key feature — when a user views a movie, the application queries the API for similar titles based on genre, director, and audience overlap. The recommended films are displayed in a horizontal row beneath the detail view, making it easy for the user to continue discovering content without returning to the main browse screen. This mirrors the "You might also like" experience familiar from streaming platforms, implemented directly through the Movie API's recommendation endpoint.

The Recommendations Engine and threaded API calls are the technical centrepieces — live data fetched asynchronously in background threads keeps the UI smooth and responsive, while the recommendation system mirrors a real streaming-platform experience, all built natively in Python with Tkinter.

>> CHALLENGES

Building a responsive GUI with Tkinter that handles asynchronous API requests without freezing the interface required implementing threading — API calls run in background threads while the UI remains interactive. Loading and displaying movie poster images from URLs inside a Tkinter canvas, managing image caching to avoid repeated downloads, and keeping the grid layout dynamically resizable were all non-trivial UI challenges in a framework not designed for rich media display.

>> OUTCOME

The Movie Application demonstrates Python GUI development, REST API integration, threading for async operations, and practical application design — delivering a fully functional desktop product that provides real entertainment value and showcases Python beyond data science into application development.

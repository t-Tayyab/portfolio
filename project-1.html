STYLESMART

A full-stack e-commerce fashion platform with real-time weather-powered outfit intelligence, a drag-and-drop Styler, location-based Outfit Finder, complete shopping flow, and Admin Panel.

HTMLCSSJavaScriptWeather APIProduct APIE-Commerce

>> PROJECT_OVERVIEW

StyleSmart is a feature-rich, full-stack e-commerce fashion platform built entirely with HTML, CSS, and JavaScript on the frontend, with a real-time product API and a live Weather API integrated to deliver a genuinely intelligent shopping experience. The platform goes far beyond a standard online store — it merges fashion retail with smart outfit intelligence, giving users personalised styling recommendations driven by real-world weather data wherever they are located.

The Shop Page is the core retail hub of StyleSmart. Products are fetched dynamically via a product API, ensuring the catalogue is always up to date. Users can filter by category, size, price range, and style type. Each product card includes quick-view details, an add-to-cart action, and a save-to-wardrobe option for use in the Styler.

The Styler Page is the centrepiece feature of the platform. Users can drag and drop clothing items — tops, bottoms, shoes, and accessories — onto a virtual outfit canvas to build their perfect look. Once an outfit is assembled, the Styler cross-references the live Weather API using the user's detected or entered location and evaluates whether the outfit is weather-appropriate. For example, a heavy wool coat might be rated excellent for a 5°C winter morning but flagged as unsuitable for a 32°C summer afternoon. The system provides a weather-match score, a suitability label (Perfect Match, Moderate, Not Recommended), and a brief explanation of why.

The Outfit Finder Page takes the concept further — instead of building an outfit manually, the user simply enters their location or allows the site to auto-detect it. The Weather API fetches the current conditions (temperature, humidity, rainfall, wind), and the system automatically surfaces recommended outfit combinations from the product catalogue that best suit those exact conditions. It filters by season, temperature range, and weather type (sunny, rainy, cold, hot) and presents ready-to-buy outfit sets.

The platform features a complete shopping flow — product listing, individual product pages, size selection, an Add to Cart system with quantity management, a Cart Page with running totals and item removal, and a full Order Page with order summary, shipping details entry, and order confirmation. The Admin Panel provides a secure backend dashboard where administrators can add new products, manage inventory, update pricing, view placed orders, and monitor the product catalogue — making it a fully operational e-commerce system.

>> CHALLENGES

The primary technical challenge was synchronising live weather data with the product recommendation engine in real time. Weather conditions can change rapidly, so the integration needed to be efficient — caching API responses to avoid excessive calls while still delivering accurate, up-to-date recommendations. Ensuring the weather matching logic was nuanced enough to handle edge cases (humidity combined with temperature, for example) required iterative testing across different climate scenarios. Building the drag-and-drop Styler interface with smooth, intuitive behaviour across both desktop and mobile also required significant CSS and JavaScript work.

>> OUTCOME

StyleSmart delivers a shopping experience that competitors simply don't offer — the combination of a live weather engine with outfit intelligence transforms a standard product catalogue into a smart personal stylist. The Admin Panel makes the platform scalable and manageable, and the clean, responsive UI ensures a premium experience on all screen sizes. This project demonstrates full-stack frontend capability, API integration, real-time data handling, and complex UX design all in a single cohesive product.

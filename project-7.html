FASHION-MNIST CLASSIFIER

A complete ML pipeline classifying 10 fashion categories from the Fashion-MNIST dataset — featuring full EDA, three trained models (best: Random Forest 88.01%), per-class analysis, and a live Gradio interactive app.

PythonScikit-learnGradioPandasNumPyKaggleMachine Learning

>> PROJECT_OVERVIEW

Fashion-MNIST Image Classification is a comprehensive machine learning project hosted as a reproducible Kaggle notebook. The project tackles the multi-class image classification problem using the Fashion-MNIST dataset — 70,000 grayscale 28×28 images across 10 clothing categories — progressing through a full end-to-end ML pipeline from exploratory data analysis to a live interactive Gradio web application where users can test the trained model in real time.

The notebook follows a rigorous, structured methodology: Section 1 imports all required libraries (NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, Gradio). Section 2 loads the Fashion-MNIST dataset and confirms structure — 60,000 training and 10,000 test samples across 10 classes. Section 3 (EDA) performs thorough exploratory analysis: class distribution charts, sample image grids per category, pixel intensity distributions, and heatmaps revealing visual patterns across the dataset.

Sections 4–6 handle the data pipeline: pixel normalisation (scaling 0–255 to 0–1), flattening the 28×28 image matrices into 784-feature vectors for the classical ML models, and performing a stratified 80/20 train-test split across all 10 classes to ensure balanced representation in both sets.

Section 7 is the modelling core — three distinct algorithms are trained and evaluated: Logistic Regression, K-Nearest Neighbors, and Random Forest. Each model is assessed with accuracy, precision, recall, and F1-score, plus a full per-class classification report across all 10 clothing categories. Charts are generated for each model's confusion matrix and per-class performance breakdown.

The Random Forest model achieved the highest accuracy at 88.01%, outperforming both Logistic Regression (85.26%) and K-Nearest Neighbors (85.45%). The strongest-performing classes were Trouser (0.98 F1), Bag (0.97 F1), and Sandal (0.97 F1), while Shirt proved the most challenging category across all three models. Section 12 concludes with a deployed Gradio interactive app where users can upload any fashion image and receive an instant classification prediction from the trained model.

The Random Forest at 88.01% accuracy and the Gradio interactive deployment are the twin highlights — rigorous multi-model benchmarking combined with a live, usable product that takes the trained pipeline beyond a static notebook into a real-world classification tool.

>> CHALLENGES

The most analytically interesting challenge was the persistent underperformance on the Shirt class — all three models struggled with it, reflecting the genuine visual ambiguity between shirts, T-shirts, and pullovers in greyscale 28×28 pixel space. Deciding which classical ML approach to use required understanding the trade-offs: Logistic Regression is fast and interpretable, KNN is non-parametric but computationally expensive at scale, and Random Forest handles non-linearity well but requires careful hyperparameter tuning. Building the Gradio app required integrating the trained model pipeline with a preprocessing function that correctly handles new input images through the same normalisation and flattening steps used during training.

>> OUTCOME

This project delivers a fully reproducible, well-documented ML pipeline that takes raw image data from exploration through to a deployed interactive application. The 88.01% Random Forest accuracy on 10-class image classification demonstrates strong feature engineering and model selection skills, and the Gradio deployment shows the ability to take a trained model into a usable product rather than leaving it as a static notebook exercise.
